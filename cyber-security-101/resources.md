Resources over the Internet the help in the field:
- GitHub: has repositories with Proof-of-Concept scripts that bscly show how an exploit works (not always safe tho; repos might be malicious)
- Shodan: can display virtually anything with a public network connection
- VirusTotal: popular blue team tool that lists recognised viruses from known security vendors, useful for gathering intel
- Common Vulnerabilities and Exposues (CVE): a database on essentially all known vulnerablities w. their details, patches etc, each with a unique identifier that also applies to GitHub repositories about the same vulneribility
- Technical documentation like Linus MANual pages for terminal commands have every detail on what commands do and how they work
